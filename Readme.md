## Перемещение между "сохранениями"

Для перемещения на другую фиксацию(коммит) используется команда *git checkout*. Для этого необходимоБ как показано в прошлом пунктеБ в журнале изменеий найти необходимый коммит и его хеш(номер)Б после чего в терминале с папкой-репозиторием надо написать *git checkout <хеш коммита>*. После выполнения этой команды мы попадаем в состояние **detached head**, в котором никакие следующие коммиты сохранять не будут.
